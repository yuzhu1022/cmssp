• 【穴位名】

    下髎穴

  • 【功效】

    调经止带，补益下焦，强腰利湿。

  • 【主治】

    主大小便不利，肠鸣注泻，寒湿内伤，大便下血，腰不得转，痛引卵，女子下苍汁不禁，中痛引小腹急痛。

  • 【取穴方法】

    在骶部，当[定位]在骶部，当中髎下内方，适对第4骶后孔处。

  • 【穴位配伍】

    配足三里、天枢主治泄泻 配风市、昆仑主治腰痛。

  • 【按压方式】

    下髎穴最好采用叩击，将手握一个空拳，用拳背这个位置去叩击腰骶部，双手可以同时叩击，叩击的力量一定要大点，有明显的声音出现，刺激它，才可能通过骶神经孔，传递到盆腔里面去，来刺激里面的脏腑。
