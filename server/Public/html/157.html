• 【穴位名】

    胆俞穴

  • 【功效】

    外散胆腑之热。

  • 【主治】

    主头痛，振寒汗不出，腋下肿胀，口苦舌干，咽痛干呕吐，骨蒸劳热食不下，目黄。

  • 【取穴方法】

    在背部，当第10胸椎棘突下，旁开1.5寸。

  • 【穴位配伍】

    胆俞配阳陵泉、太冲，有舒肝理气和胃的作用，主治呕吐，胃炎，胆道蛔虫； 胆俞配日月，为俞募配穴法，有疏肝利胆，清热除湿的作用，主治黄疸，胆囊炎； 胆俞配膏肓、三阴交，有养阴清热健脾的作用，主治咽痛，肺痨，潮热。

  • 【按压方式】

    按摩肝俞穴治疗慢性肝炎，按压肝俞穴时，一面吐气一面用力按压6秒钟，每回按压5次，每天5回。
