• 【穴位名】

    复溜穴

  • 【功效】

    补肾益阴，温阳利水

  • 【主治】

    主肠澼，腰脊内引痛，不得俯仰起坐，目视䀮䀮，善怒多言，舌干，胃热，虫动涎出，足痿不收履，胻寒不自温，腹中雷鸣，腹胀如鼓，四肢肿，五肿水病，青、赤、黄、白、黑，青取井、赤取荥，黄取输，白取经，黑取合。血痔，泄后肿，五淋，血淋，小便如散火，骨寒热，盗汗，汗注不

  • 【取穴方法】

    在小腿内侧，太溪直上2寸，跟腱的前方。

  • 【穴位配伍】

    配后溪穴、阴郄穴治盗汗不止； 配中极穴、阴谷穴治癃闭。

  • 【按压方式】

    座位屈膝，以拇指指腹点揉复溜穴。点揉的力度要均匀、柔和、浸透，使力气深达深层部分安排，以有酸痛感为佳。迟早各一次，每次点揉3～5分钟，两边复溜穴替换点揉。
