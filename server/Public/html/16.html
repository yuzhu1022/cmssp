• 【穴位名】

    商阳穴

  • 【功效】

    气化大肠经体内水液，向大肠经体表经脉输送高温水湿气体。

  • 【主治】

    主胸中气满，喘咳支肿，热病汗不出，耳鸣聋，寒热痎疟，口干颐颔肿，齿痛，恶寒，肩背急相引缺盆中痛，目青盲。灸三壮，左取右，右取左，如食顷立已。

  • 【取穴方法】

    在手食指末节桡侧，距指甲角0.1寸。

  • 【穴位配伍】

    配少商、中冲、关冲，有醒脑开窍的作用，主治中风，中暑； 配合谷、少商，有清热泻火的作用，主治咽喉肿痛，目赤肿痛； 配合谷、阳谷、侠溪、厉兑、劳宫、腕骨等。有发汗泻邪热的作用，主治热病汗不出； 配少商穴点刺出血治热病，昏迷。

  • 【按压方式】

    可在上下班乘公共汽车或地铁时，用食指钩住车内的扶手或吊环，或在闲暇时两手食指相钩反复牵拉，也可利用伞柄等按摩食指等。
