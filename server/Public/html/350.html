• 【穴位名】

    腰阳关穴

  • 【功效】

    祛寒除湿，舒筋活络。

  • 【主治】

    主膝外不可屈伸，风痹不仁，筋挛不行。

  • 【取穴方法】

    在腰部，当后正中线上，第4腰椎棘突下凹陷中，约与髂嵴相平

  • 【穴位配伍】

    腰阳关配肾俞、次髎、泻委中治腰脊痛、四肢厥冷、小便频数； 配腰夹脊、秩边、承山、飞扬治坐骨神经痛、腰腿痛； 配膀胱俞、三阴交治遗尿、尿频。

  • 【按压方式】

    左手或右手握拳，以食指指关节突起部置于腰阳关穴上，先顺时针方向压揉9次，再逆时针方向压揉9次，反复作36次。意守腰阳关穴。督脉为阳经，本穴为阳气通过之关。每天按揉此穴，具有疏通阳气、强腰膝、益下元等作用。
