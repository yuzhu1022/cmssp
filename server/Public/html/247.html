• 【穴位名】

    关冲穴

  • 【功效】

    泄热、利喉舌、开窍、活血通络。

  • 【主治】

    主喉痹喉闭，舌卷口干，头痛，霍乱，胸中气噎，不嗜食，臂肘痛不可举，目生翳膜，视物不明。

  • 【取穴方法】

    无名指尺侧指甲根角旁0.1寸。

  • 【穴位配伍】

    配少商、少泽，有泄热利咽的作用，主治咽喉肿痛。 配人中、劳宫，有泄热开窍的作用，主治中暑。 配风池、商阳，有退热解表的作用，主治热病无汗。

  • 【按压方式】

    操作时用指甲或者牙签掐关冲穴，每次掐10秒，放松2秒后重复掐按，每侧手指掐按5次。掐按时用力要均匀，使穴位能够感到微微酸痛。
