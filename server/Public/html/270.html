• 【穴位名】

    风市穴

  • 【功效】

    舒筋活络，祛风化湿，通经活络。

  • 【主治】

    主中风腿膝无力，脚气，浑身搔痒，麻痹，厉风疮。

  • 【取穴方法】

    在大腿外侧部的中线上，当腘横纹上7寸。或直立垂手时，中指尖处。

  • 【穴位配伍】

    配风池穴、大杼、大椎穴、命门穴、关元穴、腰阳关穴、十七椎治中心型类风湿。

  • 【按压方式】

    自然站立，双脚分开与肩同宽，双臂自然下垂，掌心朝内侧，中指指尖紧贴风市穴，拔顶，舌抵上腭，提肛，净除心中杂念。两手掌相互摩擦至热，两手心对正胃部，距离约10厘米，十个脚趾同时抓地，每次做lO分钟。每次饭后l小时做此功。效果显著。适应病症：消化不良。
