• 【穴位名】

    率谷穴

  • 【功效】

    有疏风活络、镇惊止痛的作用。

  • 【主治】

    主痰气膈痛，脑两角强痛，头重，醉后酒风，皮肤肿，胃寒，饮食烦满，呕吐不止。

  • 【取穴方法】

    在头部，当耳尖直上入发际1.5寸，角孙直上方。

  • 【穴位配伍】

    率谷配风池、太阳，有祛风止痛的作用，主治偏头痛； 率谷配人中、曲池、太冲，有祛风清热镇惊的作用，主治小儿惊风； 率谷配足三里、中脘，有和胃止呕的作用，主治呕吐。

  • 【按压方式】

    减轻偏头痛需按摩率谷穴。找到穴位后，分别以两手中指指腹按压在穴位上，按10～15 分钟，以患者头痛有明显减轻为度。
