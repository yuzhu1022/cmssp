• 【穴位名】

    大钟穴

  • 【功效】

    益肾平喘，调理二便。

  • 【主治】

    主呕吐，胸胀喘息，腹满便难，腰脊痛，少气，淋沥洒淅，腹脊强，嗜卧，口中热，多寒，欲闭户而处，少气不足，舌干、咽中食噎不得下，善惊恐不乐，喉中鸣，咳唾气逆，烦闷。实则闭癃泻之，虚则腰痛补之。

  • 【取穴方法】

    在足内侧，内踝下方，当跟腱附着部的内侧前方凹陷处。

  • 【穴位配伍】

    配行间穴缓解治疗虚火上炎之易惊善怒； 配鱼际穴缓解治疗虚火上炎之咽痛； 配太溪穴、神门穴缓解治疗心肾不交之心悸、失眠。

  • 【按压方式】

    按揉大钟穴和天柱30～50次，也可用指腹按住此处6秒钟，然后慢慢松开，如此反复按压，不拘时做。
