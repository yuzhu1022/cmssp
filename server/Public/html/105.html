• 【穴位名】

    灵道穴

  • 【功效】

    安神，宁心，通络。

  • 【主治】

    主心痛，干呕，悲恐，相引瘈疭，肘挛，暴喑不能言。

  • 【取穴方法】

    在前臂掌侧，当尺侧腕屈肌腱的桡侧缘，腕横纹上1.5寸。

  • 【穴位配伍】

    配内关、巨阙、中冲治心绞痛； 配人中、合谷、巨阙治癔病；配天突、天窗治暴瘖；配尺泽、少海治肘挛； 灵道配外关，有舒筋通络活血的作用，主治臂痛，指麻，关节炎； 灵道配廉泉，有利舌启闭的作用，主治舌强，暴暴瘖，癔病； 灵道配郄门，安神宁心，镇静止痛。

  • 【按压方式】

    冠心病犯病时，可用拇指先轻揉灵道穴1分钟，然后重压按摩2分钟，最后轻揉1分钟，每天上下午各揉1次，10天为一疗程，间歇2～3天，可进行下一疗程。经观察，揉按治疗后心绞痛症状明显减轻，心电图亦有改善。
