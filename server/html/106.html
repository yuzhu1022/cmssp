• 【穴位名】

    阴郄穴

  • 【功效】

    沟通心肾。

  • 【主治】

    主鼻衄吐血，洒淅畏寒，厥逆气惊，心痛霍乱，胸中满。

  • 【取穴方法】

    在前臂掌侧，当尺侧腕屈肌腱的桡侧缘， 腕横纹上1.5寸。

  • 【穴位配伍】

    本穴配内关、心俞治心痛、配百劳、肺俞、定喘治肺结核、配后溪治盗汗； 阴郄配心俞、神道，有通阳行气，宁心定悸的作用，主治心痛，心悸，神经衰弱； 阴郄配尺泽、鱼际，有清热凉血止血的作用，主治衄血、吐血； 阴郄配后溪、三阴交，有清虚热，敛阴液的作用，主治阴。

  • 【按压方式】

    用手指指腹按压阴郄穴位，按摩时要注意力度适中，每次按摩5分钟，每天按摩2次。特别注意：每日经常按摩阴郄穴位可以预防和治疗心脏疾病、小儿抽筋等。
