• 【穴位名】

    口禾髎穴

  • 【功效】

    开关通窍。

  • 【主治】

    主尸厥及口不可开，鼻疮息肉，鼻塞不闻香臭，鼽衄不止。

  • 【取穴方法】

    在上唇部，鼻孔外缘直下，平水沟穴。

  • 【穴位配伍】

    鼻塞，不辨香臭：口禾髎、迎香、上星、五处、水沟、风府、百劳、太渊 息肉：口禾髎、风池、风府、迎香、人中 鼻衄：口禾髎、上星、合谷、鱼际、少商 口噤不开：口禾髎、合谷、列缺、颊车 尸厥：口禾髎、攒竹

  • 【按压方式】

    用食指指腹按压口禾髎，每次5-10分钟，以有酸痛感为宜，对过敏性鼻炎、鼻前庭炎和慢性鼻炎均有较好疗效。
