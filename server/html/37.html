• 【穴位名】

    人迎穴

  • 【功效】

    接收胃经气血并分流胸腹。

  • 【主治】

    主吐逆霍乱，胸中满，喘呼不得息，咽喉痈肿，瘰疬。

  • 【取穴方法】

    在颈部，喉结旁，当胸锁乳突肌的前缘，颈总动脉搏动处。

  • 【穴位配伍】

    配大椎穴、太冲穴治高血压。

  • 【按压方式】

    坐位或仰卧，用拇指指腹在一侧人迎穴向内下方缓慢用力按压，至手下有明显搏动感觉，按5-8秒，然后将手指迅速放开，此时头部会有发热的感觉，休息15-20秒后，待发热感觉退却后，再重复操作两次；然后用手指轻轻弹拔胸锁乳突肌30-40次。再换另一侧操作，每日1-2次。
