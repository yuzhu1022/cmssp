• 【穴位名】

    曲骨穴

  • 【功效】

    通利小便，调经止痛。

  • 【主治】

    主失精，五脏虚弱，虚乏冷极，小腹胀满，小便淋涩不通，㿉疝，小腹痛，妇人赤白带下。

  • 【取穴方法】

    在下腹部，当前正中线上，耻骨联合上缘的中点处。

  • 【穴位配伍】

    配肾俞、志室、大赫、关元、命门治阳萎、遗精（肾气虚型）； 配膀胱俞、肾俞、次髎、阴陵泉、蠡沟治阳萎、遗精、癃闭、淋症、阴痒、湿疹、带下（湿热下注）； 配中极、关元、肾俞治肾虚、遗尿、小便不利； 配关元、命门、阴交（针补法或灸）治宫寒不孕、痛经。

  • 【按压方式】

    每天按摩曲骨穴50～100次，可以很好地缓解前列腺的压力，解决尿频尿急等小便问题。需要注意的一点就是，这个穴离膀胱很近，所以，最好排空小便再来按摩，力度可以相对大一点，刺激到位。
