• 【穴位名】

    膈俞穴

  • 【功效】

    理气宽胸，活血通脉。

  • 【主治】

    主心痛，周痹，吐食翻胃，骨蒸，四肢怠惰，嗜卧，痃癖，咳逆，呕吐，膈胃寒痰，食饮不下，热病汗不出，身重常温，不能食，食则心痛，身痛肿胀，胁腹满，自汗盗汗。

  • 【取穴方法】

    在背部，当第7胸椎棘突下，旁开1.5寸。

  • 【穴位配伍】

    配内关、足三里治呕吐、呃逆； 配足三 里、血海、肓膏治贫血。

  • 【按压方式】

    两手置于被施术者上背部，双手拇指指腹分别按揉两侧的膈俞穴。按揉的手法要均匀、柔和，以局部有酸痛感为佳。早晚各1次，每次按揉2～3分钟，两侧膈俞穴同时按揉。
