• 【穴位名】

    少海穴

  • 【功效】

    理气通络，益心安神，降浊升清。

  • 【主治】

    主寒热齿龋痛，目眩发狂，呕吐涎沫，项不得回顾，肘挛腋胁下痛，四肢不得举，齿寒，脑风头痛，气逆噫哕，瘰疬，心疼，手颤健忘。

  • 【取穴方法】

    屈肘，当肘横纹内侧端与肱骨内上髁连线的中点处。

  • 【穴位配伍】

    配合谷、内庭，有清泻阳明热邪的作用，主治牙痛，牙龈肿痛； 配后溪，有舒筋通络活血的作用，主治手颤，肘臂疼痛； 配天井，有活血散瘀的作用，主治瘰疬。

  • 【按压方式】

    在心火较旺的时候每天可以按摩3-4次，每次1-2分钟就可以了。每天的11～13点，也就是人体阳气最足的时候去搓心经，特别是肘关节内侧的少海穴要搓热，补肾的效果好。
