• 【穴位名】

    脑空穴

  • 【功效】

    降浊分清。

  • 【主治】

    主劳疾羸瘦，体热，颈项强不可回顾，头重痛不可忍，目瞑心悸，发即为癫风，引目眇，鼻痛。魏武帝患头风，发即心乱目眩，华佗针脑空立愈。

  • 【取穴方法】

    在头部，当枕外隆凸的上缘外侧，头正中线旁开2.25寸，平脑户。

  • 【穴位配伍】

    配大椎穴、照海穴、申脉穴治癫狂痫证； 配风池穴、印堂穴、太冲穴治头痛、目眩； 配悬钟穴、后溪穴治颈项强痛。

  • 【按压方式】

    坐位，用双手拇指罗纹面分别按揉两侧脑空穴半分钟，以酸胀为宜，其他手指置于旁边以助力。该手法可防治头痛、目眩、颈项强痛等病症。脑空穴大致位于头后枕外隆突的上缘，距正中线约三横指。
