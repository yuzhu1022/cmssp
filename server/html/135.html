• 【穴位名】

    心俞穴

  • 【功效】

    散发心室之热。

  • 【主治】

    主偏风半身不遂，心气乱恍惚，心中风，偃卧不得倾侧，汗出唇赤，狂走发痫，语悲泣，心胸闷乱，咳吐血，黄疸，鼻衄，目瞤目昏，呕吐不下食，健忘，小儿心气不足，数岁不语。

  • 【取穴方法】

    在背部，当第5胸椎棘突下，旁开1.5寸。

  • 【穴位配伍】

    配巨阙主治心痛； 配脾俞、神门、足三里、三阴交主治失眠健忘； 配大椎主治癫痫。

  • 【按压方式】

    以一手手掌置于心俞穴(背部第五胸椎棘突下，向两侧旁开1.5寸)进行揉法，以顺时针为主，反复3-5分钟后，再揉另一侧，力度要轻柔。
